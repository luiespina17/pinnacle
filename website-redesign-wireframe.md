## 5) Desktop + Mobile Layout Notes (Implementation)

## Global responsive rules
- Max content width: 1200px
- Section padding:
  - Desktop: 80px top/bottom
  - Tablet: 56px top/bottom
  - Mobile: 40px top/bottom

## Header
- Desktop: full nav + primary CTA visible.
- Mobile: logo + hamburger + sticky bottom bar (Call / Book).

## Hero
- Desktop: 2 columns (text left, image right).
- Mobile: stack with text first, CTAs immediately visible before image.
- Keep hero headline under ~2 lines mobile.

## Cards and grids
- Treatments: 3 columns desktop, 2 tablet, 1 mobile.
- Conditions: 4 columns desktop, 2 tablet, 1 mobile.
- Ensure equal card heights and consistent button alignment.

## Testimonials
- Desktop: 3 cards.
- Mobile: slider with 1 card per view.

## Map section
- Desktop: map left / info right.
- Mobile: info first, map second (to prioritize conversion actions).

## Buttons
- Minimum touch size: 44px height mobile.
- Keep max 2 buttons in a section.

## 6) Simple Style Guide (Modern, Premium, Less Hospital)

## Color system (map to logo tones in Kadence global palette)
- **Primary brand color:** use logo primary for buttons/links.
- **Deep neutral (headings/nav):** #1F2A37
- **Light background:** #F7FAFC
- **Surface white:** #FFFFFF
- **Body text:** #374151
- **Muted text:** #6B7280
- **Accent color (sparingly):** #14B8A6

### Usage ratio
- 70% neutral/light backgrounds
- 20% deep text and structural contrast
- 10% brand/accent highlights

## Typography
- Heading font: **Poppins** (600/700)
- Body font: **Inter** (400/500)
- H1: 46/54 desktop, 34/42 mobile
- H2: 34/42 desktop, 28/36 mobile
- Body: 18/30 desktop, 16/28 mobile
- Button text: 16px semibold

## Spacing + shape
- 8px spacing scale (8,16,24,32,40,56,80)
- Card radius: 14px
- Button radius: 999px (pill)
- Soft shadow only: 0 8px 24px rgba(15,23,42,.08)

## Image direction
- Use authentic clinic/team imagery over generic stock.
- Prioritize movement, recovery, provider-patient interaction.
- Keep overlays light; avoid dark, heavy “hospital” aesthetic.
